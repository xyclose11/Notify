# Notify

### An all-in-one note management app.
With this app you can do the following:
- Take notes
- Download notes
- **Create** & **Manage** an account
- Group notes
- Tag notes

The motivation behind this project is to learn C#, along with the .NET framework.

#### Technologies Used:
- .Net 8
- ASP.NET MVC
- JQuery
- Bootstrap
